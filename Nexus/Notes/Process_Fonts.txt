
1. Right-click "Content.mgcb" and "Open as" Monogame.

2. Create the Sprite Font file. It should be something like MyFont.spritefont
	- Note: Can only use fonts that are saved in C:/Windows/Fonts (or whatever the fonts folder is)

3. Save, and Build.

4. Make sure /Content/Fonts/MyFont.spritefont properties are marked as:
	Build Action: Content
	Copy to Output Directory: Copy if Newer
